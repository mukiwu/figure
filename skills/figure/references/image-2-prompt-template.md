- Include `pure white background`.
- Include the rabbit's action as a verb phrase.
- Include the preferred reference proportion rule: soft oval head only slightly larger than the torso, small shirt-shaped body, short thin arms and legs, light wobbly posture, no oversized toddler head.
- Include the character consistency lock with the preferred reference proportions and tiny off-center short horizontal dash mouth.
- Include compact action constraints: short arms, bent elbows, hands close to the torso, target object close to the rabbit, no stretched writing hand.
- Include the visually distinct face rule — natural uneven ears, tiny uneven dot eyes, small subtle nose, tiny off-center short horizontal dash mouth, no x-shaped mouth, no smile curve, no blush, no cheek marks.
- Include the instruction to keep the character visually distinct from Miffy / Nijntje and similar classic minimalist picture-book rabbits.
- Include the muted clothing color block rule with the thin white gap.
- Include exact text labels, usually 2-5 only, in the source language by default.
- Include the instruction that every label character must be reproduced exactly and legibly, with no garbled or substituted glyphs; prefer fewer Chinese characters per label.
- Include the exact label style sentence.
- Include the moderate diagram marks density instruction.
- Include a negative instruction that rejects decorative IP usage.
- Include a negative instruction against instruction-manual layouts and overpacked information.
